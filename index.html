Step-By-Step Go-To-Guide

Ordered List: Example Workflow

(If there is a cloned repo, start on step 3)

  1. Create repo on GitHub
  2. Clone Repo - git clone [url]
  3. Go to folder - cd [repo-folder]
  4. Create Branch - git checkout -b [new-branch]
  5. (Optional)Create file - touch [filename]
  6. Edit file - subl [filename]
  7. Stage file - git add [filename](tip: git add --all)
  8. Commit file - git commit -m "commit message"
  9. Send changes to GitHub - git push origin [branch-name]
  10. Go to GitHub, review and merge there
  11. Back in terminal, go to master - git checkout master
  12. Pull changes from GitHub - git pull origin master

TIP:You can do a initial commit on master with git push -u origin master and at the end use a simple git push to pull changes.
